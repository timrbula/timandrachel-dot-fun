# Pictures Directory

Add your 10 wedding pictures here!

## File Naming

The pictures page expects the following filenames:

- pic1.jpg
- pic2.jpg
- pic3.jpg
- pic4.jpg
- pic5.jpg
- pic6.jpg
- pic7.jpg
- pic8.jpg
- pic9.jpg
- pic10.jpg

## Supported Formats

You can use:

- `.jpg` or `.jpeg`
- `.png`
- `.gif`
- `.webp`

Just make sure to update the filenames in `/src/pages/pictures.astro` if you use different names or formats!

## Tips

- Images will be displayed in a single column
- They'll automatically resize to fit the page width
- For best results, use landscape or square images
- Recommended width: at least 800px for good quality
